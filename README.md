# 💎 Гроші — фінансовий застосунок

Особистий облік доходів і витрат у гривнях. Працює як застосунок на iPhone (PWA), офлайн, з PIN-захистом, неоновим інтерфейсом, календарем, діаграмами та фінансовою мудрістю на головному екрані.

---

## 📲 Як встановити на iPhone (через GitHub Pages)

> Спосіб надійний: працює офлайн, дані зберігаються на телефоні.

### Крок 1. Завантажити файли на GitHub
1. Зайди на https://github.com → **Sign up / Log in** (акаунт безкоштовний).
2. Натисни **New repository** (зелена кнопка / «+» вгорі праворуч).
   - Repository name: `groshi` (будь-яка назва)
   - Обери **Public**
   - Натисни **Create repository**.
3. На сторінці репозиторію натисни **uploading an existing file**
   (або **Add file → Upload files**).
4. Перетягни **усі файли з цієї папки** (окрім папки `чеки` — вона не потрібна на сайті):
   - `index.html`, `styles.css`, `app.js`, `data.js`
   - `manifest.webmanifest`, `sw.js`
   - папку `icons` (з усіма картинками)
5. Натисни **Commit changes**.

### Крок 2. Увімкнути GitHub Pages
1. У репозиторії: **Settings → Pages** (ліве меню).
2. У розділі **Branch** обери `main` і теку `/ (root)` → **Save**.
3. Зачекай 1–2 хвилини. З'явиться адреса виду:
   `https://ТВІЙ_ЛОГІН.github.io/groshi/`

### Крок 3. Додати на екран «Домой»
1. На iPhone відкрий цю адресу **в Safari** (саме Safari, не Chrome!).
2. Натисни кнопку **«Поділитися»** (квадрат зі стрілкою вгору).
3. Обери **«На екран Домой»** → **Додати**.
4. Готово — іконка 💎 з'явиться на робочому столі.
   Відкривається на весь екран, без браузера, працює офлайн.

### Крок 4. Перший запуск
- Застосунок попросить **створити PIN-код** (4 цифри) — введи двічі.
- Далі при кожному вході він питатиме цей код.

---

## 🧾 Як додавати чеки минулих/поточних періодів

1. Кидай фото чеків у папку **`чеки`** (на комп'ютері).
2. Скажи мені: **«розпізнай чеки в папці»**.
3. Я зчитаю суми, дати й категорії та зроблю файл `groshi-import.json`.
4. У застосунку: **Ще → Імпорт даних** → обери цей файл.
   Операції з'являться в обліку, календарі та діаграмах.

Можна також додавати операції **вручну** — кнопка **«+»** унизу.

## ✨ Що вміє застосунок

| Розділ | Можливості |
|--------|-----------|
| **Головна** | Баланс за місяць, доходи/витрати, остання активність, фінансова мудрість (нова при кожному вході) |
| **Календар** | Місячна сітка з мітками доходів (🟢) і витрат (🔴), підсумок дня |
| **+ Додати** | Дохід/витрата, сума, 24 статті (+ власні), дата, нотатка |
| **Аналітика** | Період: день / тиждень / місяць / рік / весь час / довільний. Кільцева діаграма за статтями + порівняння з попереднім періодом |
| **Ще** | Зміна PIN, експорт/імпорт (резервна копія), власні статті, очищення |

- 💾 Усі дані зберігаються **локально на телефоні** (нікуди не передаються).
- 📴 Працює **офлайн** після першого відкриття.
- 🇺🇦 Повністю українською, валюта — **гривня (₴)**.

## 🔄 Резервні копії
**Ще → Експорт даних** збереже файл `groshi-backup-ДАТА.json`.
Його можна імпортувати назад у будь-який момент (або перенести на інший пристрій).

---

## 🛠️ Запуск локально (для перевірки на комп'ютері)
```
python -m http.server 8000
```
Відкрий у браузері: `http://localhost:8000`

> ⚠️ PWA (офлайн + іконка) повноцінно працює лише через `https://` (GitHub Pages),
> тому для встановлення на iPhone використовуй спосіб вище.
